# Positioning

Bad positioning:

> Rosetta Envelope ends OCR and lets AI understand all documents.

Better positioning:

> Rosetta Envelope helps newly generated business documents carry a signed, structured payload so downstream software can read fields directly instead of reconstructing them from layout or OCR.

## Best initial use cases

- exam papers
- delivery notes
- purchase orders
- internal expense forms
- generated invoices outside regulated e-invoicing flows

## Not ideal initial use cases

- arbitrary historical scans
- legal evidence workflows
- regulated e-invoicing replacement
- universal PDF parsing
- high-security production signing
