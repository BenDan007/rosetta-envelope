# Contributing

This project is early and experimental.

Useful contributions include:

- clearer examples
- JSON Schema design
- QR payload size improvements
- PDF metadata extraction
- real signing design
- scanner demos
- documentation fixes

Please keep the project grounded. Avoid claims such as:

- "ends OCR"
- "100% understands all documents"
- "stores complete semantics"
- "global standard"

The goal is small, testable, and practical.
